Personal website and online resume of **Ki-Ha Kim** — Ph.D., Senior Researcher at KISTI.

Research focus: high-performance computing (HPC), computational fluid dynamics (CFD),
and GPU-accelerated parallel numerical solvers (multi-GPU tridiagonal / sparse linear solvers, CUDA).
